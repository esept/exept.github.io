方法的重载
多个方法的名称一样，但是参数列表不一样
相关：
参数个数不同/参数类型不同/参数的多类型顺序不同
无关：

数组

数组是一种引用数据类型
数组当中的多个数据，类型必须统一
数组的长度在程序运行期间不能改变

数组的初始化：
在内存中创建一个数组，并向其中赋予一些默认值
动态初始化：指定长度

动态初始化数组的格式：

数据类型[] 数组名称 = new 数据类型[数组长度]

左侧数据类型，代表数组中保存的数据，全是统一的什么类型
【】-> 表示这是一个数组
数组名称-> 数组的名字
<=>
new 代表创建数组的动作
数据类型： 必须和左边的数据类型保持一致
中括号的长度： 数组中保存的数据个数，是一个int数字

静态初始化：指定内容:直接将具体的数据内容进行指定
数据类型[] 数组名称 = new 数据类型[] {元素1,元素2···}

使用静态初始化数组的时候，格式还可以在省略一下：可以省略’new 数据类型[]’

初始化的拆分

访问数组元素
直接打印数组名称得到的是数组对应的内存地址哈希值

访问数组的格式：
数组名称[索引值]

在使用动态初始化数组的时候，其中的元素都会自动有一个默认值
int：0
float：0.0
字符类型：‘\u0000’
布尔类型：false
如果是引用类型：null
